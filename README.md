Tab Switching Interface
A simple, responsive tab interface built with HTML, CSS, and Vanilla JavaScript. It enables users to switch between content sections dynamically without reloading the page.

Features
Dynamic tab switching
Clean and minimal UI
No external libraries required
Easy to customize and extend

How It Works
Each tab is linked to a content section using a data-tab attribute. When a tab is clicked, the script updates the active class to display the corresponding content while hiding others.

Tech Stack
HTML
CSS
JavaScript

Usage
Clone the repository and open index.html in your browser.
